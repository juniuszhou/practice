Overview

Example program that simulates a firewall.

Files

PacketProcessing.cpp
Source code for the example.

Directories

win/vc7.1
Contains Microsoft* Visual Studio* .NET 2003 workspace for building and running the example.
win/vc8
Contains Microsoft* Visual Studio* 2005 workspace for building and running the example.
linux
Contains Linux* Makefile for building and running the example.
mac/xcode
Contains Mac OS* Xcode* workspace for building and running the example.

Usage

PacketProcessing [infile] [outfile]
The infile parameter is the input file containing packet data. It defaults to a platform-dependent value which allows the program to run out of the box from the IDE. The outfile parameter specifies the output file where the program writes its data. It defaults to output.txt.
