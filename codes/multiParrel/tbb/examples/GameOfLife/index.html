Overview

Parallel implementation of the Conway's Game of Life.

Note: This sample is written in C++/CLI, which is only available under Microsoft* Visual Studio* 2005.

Directories

src
Source code implementing the Game of Life.
win/vc8
Contains Microsoft* Visual Studio* 2005 workspace for building and running the example.
